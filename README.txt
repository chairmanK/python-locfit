python-locfit is intended to be a Python wrapper around the
Locfit software system by Catherine Loader

Project goals:
- Fully expose the power of core Locfit C routines
- Provide Pythonic interfaces and abstractions
- Maintain compatibility with NumPy and follow NumPy conventions
- Eventually become a part of the SciPy project
